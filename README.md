# AI Values Discovery Assistant

An AI-powered web application that helps users identify their core values and generate a polished personal values statement through guided reflection.

---

## Overview

The AI Values Discovery Assistant supports users in articulating what matters most to them. By combining structured reflection prompts with large language model (LLM) synthesis, the app transforms personal input into a clear, professional values statement.

This project was developed as part of an AI Ethics course and emphasizes human-centered AI design.

---

## Features

- Guided value selection workflow  
- Narrowing down to top 5 core values  
- Reflection-based input (behavioral and decision-oriented)  
- AI-generated personal values statement  
- Clean, readable output formatting  

---

## Tech Stack

- Python  
- Streamlit  
- Hugging Face Inference API  
- LLM: Llama 3.1 (chat-based inference)  

---

## How It Works

1. Select values that resonate  
2. Narrow to top 5 core values  
3. Reflect on meaning, decisions, and behaviors  
4. Generate a structured personal values statement  

---

## AI Design Considerations

This project focuses on human-AI collaboration:

- Users provide meaningful input (human-in-the-loop)  
- The model synthesizes rather than replaces thinking  
- Prompt design guides structure, tone, and quality  
- Output remains grounded in user input
